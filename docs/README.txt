FuzzyART Documentation Directory
=================================

This directory contains the complete Sphinx documentation for FuzzyART.

DIRECTORY STRUCTURE:
- index.rst          : Main documentation index
- readme.rst         : Project overview
- installation.rst   : Installation guide
- quickstart.rst     : Quick start tutorial
- changelog.rst      : Version history
- contributing.rst   : Contributing guidelines
- guide/             : User guides (algorithm, preprocessing, hyperparameters)
- examples/          : Working examples and tutorials
- api/               : API reference documentation
- _static/           : Static files (images, stylesheets, etc.)
- _build/            : Built documentation (generated by Sphinx)
- conf.py            : Sphinx configuration file (not included, use your own)
- Makefile           : Make commands for building

TO BUILD THE DOCUMENTATION:
1. Ensure Sphinx is installed: pip install sphinx sphinx-rtd-theme
2. Run: make clean html
3. Open: _build/html/index.html in your browser

For more information, see the README.md in the root of the package.

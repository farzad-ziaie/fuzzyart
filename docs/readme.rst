About FuzzyART
==============

FuzzyART is a Python implementation of Fuzzy Adaptive Resonance Theory (ART) and related neural network models.

This documentation covers:

- **Installation**: How to install and set up FuzzyART
- **Quick Start**: Getting started with your first classification
- **User Guide**: In-depth guides on algorithms, preprocessing, and hyperparameters
- **Examples**: Practical examples demonstrating FuzzyART capabilities
- **API Reference**: Complete API documentation

For more information, visit the GitHub repository or consult the main README file.

Features
--------

FuzzyART provides implementations of several adaptive resonance theory models:

- Fuzzy ART
- Fuzzy ARTMAP
- Bayesian ARTMAP
- Semi-supervised ARTMAP
- Ensemble methods

All models support:

- Online learning
- Batch training
- Scikit-learn compatible interface
- Flexible preprocessing options
